Read me bandeau_institutionsHere is a small script to create a banner, which can be used for instance at the bottom of a page, grouping together the logos (jpg or png) of the institutions supporting a project. All logos are scaled to the same size and neatly aligned. The sizes can be adjusted in the code.1. Création de l'environnement virtuelbash # Créer un environnement virtuel nommé "bandeau_env"python -m venv bandeau_env2. Activation de l'environnementSur Windows :bash # PowerShellbandeau_env\Scripts\Activate.ps1# Command Promptbandeau_env\Scripts\activate.batSur macOS/Linux :bash source bandeau_env/bin/activate3. Installation des dépendancesbash # Une fois l'environnement activé (vous verrez "(bandeau_env)" dans le terminal)pip install Pillow4. Vérification de l'installationbash pip list# Devrait afficher Pillow et ses dépendances5. Exécution du scriptbash python script.py6. Désactivation de l'environnementbash deactivateStructure complŹte du projetmon_projet_bandeau/??? bandeau_env/          # environnement virtuel (créé automatiquement)??? script.py             # votre code Python??? logo1.png            # logos ą assembler??? logo2.jpg??? requirements.txt     # optionnel (voir ci-dessous)??? bandeau_institutions.jpg  # résultat généréBonus : Fichier requirements.txtCréez un fichier requirements.txt :Pillow==10.1.0Puis installez avec :bash pip install -r requirements.txtCela facilite le partage et la reproduction de l'environnement.
